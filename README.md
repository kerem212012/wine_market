# Новое русское вино

Сайт магазина авторского вина "Новое русское вино".

## Запуск

- Скачайте код.
- Создать файл ```.env```.
- Написать внутрь команду ```FILE_PATH = "Файл или путь file/example.xlsx"```.
- Если файл, то скачайте в репозиторий тот файл.
- Запустите сайт командой ```python main.py -f example/.xlsx``` или вы хотите файл по умолчанию ```python main.py```
- Перейдите на сайт по адресу [http://127.0.0.1:8000](http://127.0.0.1:8000).
